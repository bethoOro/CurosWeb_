Elementos block

Los elementos en bloque van a ocupar todo el espacio de la pantalla, los elementos en bloque realiza un salto de linea.

Elementos inline

Los elementos en linia van ocupar el espacio que necesite, va ocupar el espacio del contendio que contenga y no genera salto de linea.

Etiqueta div

La etiqueta div nos permite estructura nuestro ducumento y nos permite dividir secciones y agrupar otras etiquitas, los div son como contenedores y funcionan como elementos block.

Etiqueta span

La etiqueta span es un contenedor en linea y normalmente se ocupa para seleccionar texto.

Modelo de contenido html5

Las etiquetas de modelo de contendio html 5 funciona como un div, pero ayuda a estructurar mejor una página

Header

Es el cabecero de la página

Footer

Es el pie de la página

Nav

Es la navegación

Aside

Es una barra de herramientas o navegación secundaria
